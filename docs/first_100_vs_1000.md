# First-100 vs 100-1000 vs >1000 Regimes

## Overview

| Regime | Sessions | Strategy | Max Diff |
|--------|----------|----------|----------|
| **First-100** | 0-100 | Aggressive exploration | 0.8 |
| **100-1000** | 100-1000 | Balanced | 0.4 |
| **>1000** | 1000+ | Neural-guided | 0.1 |

## First-100 Regime

*"Move fast and break things (except brand)"*

- Try aggressive changes: layout, hero images, CTA position
- Make quick decisions
- Use LLM judgment for mutations

**Allowed Changes:**
- CTA (color, text, size, position)
- Headlines, subheadlines
- Hero images
- Testimonials, form fields
- Layout, spacing

## 100-1000 Regime (Middle)

*"Trust the data, but stay nimble"*

- Lock winning patterns
- Test smaller changes
- Use ghost-memory from tombstones

**Allowed Changes:**
- CTA (color, text, size)
- Headlines
- Testimonial order
- Spacing

## >1000 Regime (Neural)

*"The model knows best"*

- Neural predictions guide changes
- Minimal mutations
- High confidence required

**Allowed Changes:**
- CTA text only
- Headlines only
- Testimonial order

## Forbidden (All Regimes)

- Logo
- Brand colors
- Legal text
- Pricing
- Privacy policy
